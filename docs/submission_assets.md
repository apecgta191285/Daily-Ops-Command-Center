# Graduation Submission Assets

This file records the non-runtime assets that are intentionally versioned for
the graduation-project submission package.

## Chapter 3

- `docs/chapter3-data-dictionary/` contains the schema truth report and data
  dictionary draft used to align the written report with the implemented
  database.
- `docs/chapter3-process-description/` contains process descriptions in draft
  and thesis-facing formats.
- `docs/chapter3-diagrams/` contains editable Draw.io sources plus exported
  SVG/PNG diagrams for the final report.

## Presentation

- `docs/presentation/` contains the slide outlines, generation scripts, and
  generated PowerPoint files for the project presentation.
- The `.cjs` scripts are kept with the slide decks so the generated decks can be
  reproduced instead of edited as untraceable one-off files.

## Excluded Local Artifacts

The repo intentionally ignores local screen captures, ad hoc audit scratch
files, and local agent/tool configuration. Those files are useful while working,
but they are not source truth for the application or the submission package.
