# System Is / Is Not for Option A
วันที่: 23 เมษายน 2026

## System Is
- internal web app สำหรับทีมดูแลห้องปฏิบัติการคอมพิวเตอร์ในมหาวิทยาลัย
- ระบบที่ใช้ `room + time scope` เพื่อจัดการงานประจำวัน
- ระบบที่เชื่อม checklist, incident, dashboard/workboard, history, และ governance เข้าด้วยกัน
- ระบบที่ช่วยให้รู้ว่าปัญหาเกิดที่ห้องไหน และถ้าจำเป็นอ้างถึงอุปกรณ์แบบ lightweight ได้

## System Is Not
- machine registry platform
- inventory system
- maintenance management suite
- enterprise multi-tenant platform
- public-signup SaaS
- production-grade operations platform

## What To Say If Asked About Machines
ใช้คำตอบนี้:

> ตอนนี้ระบบรองรับ room context เป็นหลัก และมี equipment reference แบบข้อความสั้นเพื่อช่วยระบุอุปกรณ์ที่เกี่ยวข้อง แต่ยังไม่ได้ทำ machine registry หรือ machine lifecycle อย่างเป็นระบบ

## What To Say If Asked Why Stop Here
ใช้คำตอบนี้:

> เพราะขอบเขตที่ตั้งใจทำในรอบนี้คือทำให้การปฏิบัติงานประจำวันของหลายห้องใช้งานได้จริงก่อน ถ้าจะไปถึง machine registry ต้องเปิด wave ใหม่และวาง schema กับ workflow อีกชุดอย่างเป็นระบบ
